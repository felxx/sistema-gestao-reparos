# Sistema de Gestão de Reparos Técnicos

Sistema web desenvolvido para facilitar o gerenciamento de clientes, equipamentos, ordens de serviço e estoque de peças.

## Contexto Acadêmico

Este projeto está sendo desenvolvido como requisito de avaliação para o componente curricular de **Novas Aplicações em Engenharia de Software** do curso de **Engenharia de Software** no **Instituto Federal do Paraná (IFPR)**.

## Funcionalidades (Roadmap)

* **Gestão de Cadastros (CRUD):** Inserção, alteração, exclusão e listagem [cite: 12] de Clientes, Equipamentos, Peças e Serviços.
* **Controle de Ordens de Serviço (OS):** Acompanhamento de status, datas de entrada/previsão, defeitos relatados e laudos técnicos.
* **Autenticação e Permissões:** Controle de acesso para o técnico (visão global) e clientes (visão restrita).
* **Painel do Cliente:** Acesso para que o cliente acompanhe exclusivamente o andamento de suas ordens de serviço.

## Tecnologias Utilizadas

* **Back-end:** Python e Django
* **Front-end:** HTML, CSS e JavaScript
